# MapNotepad
Mobile app for creating and managing notes on Google Maps.
<br/>
All data is stored in the Firebase Realtime Database, protected by rules.
<br/>
In development.

## Developed using
- TypeScript
- React Native
- Expo
- React Native Maps
- Redux Toolkit
- Axios
- Firebase
- Firebase Auth REST API
- React Hook Form
